About GreenPath AI

GreenPath AI is a multimedia route visualization prototype that compares two A* pathfinding routes: the fastest route and the greenest route.

How It Works

The A* algorithm searches for a path from the start point to the destination by combining:

  1. The cost already travelled
  2. An estimated cost to the destination

In this prototype, A* is run using different route costs:

  • Fastest route: prioritizes estimated travel time
  • Greenest route: prioritizes lower pollution or environmental cost
