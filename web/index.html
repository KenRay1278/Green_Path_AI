Green Path AI

Compare the fastest route and the greenest route across Greater Jakarta — powered by real OpenStreetMap roads and A* pathfinding.

Try the demo

Two ways to travel

⚡ Fastest route

Minimizes travel time using A* with time-based edge costs. Best when you need to arrive quickly.

🌱 Greenest route

Minimizes estimated pollution using road type, distance, and intersection penalties. May take longer but reduces emissions.

How it works

1

Road network

Real Jakarta streets downloaded from OpenStreetMap via OSMnx and stored as a graph.

2

Pollution weights

Each road gets a pollution score based on type (e.g. residential vs motorway) and intersections.

3

Dual A*

Two A* runs — one optimizing time, one pollution — then side-by-side stats on the map.

On the map

  • Interactive Leaflet map — click start and end points
  • Route comparison: distance, time, and pollution
  • Optional A* exploration animation
  • Dark mode and route visibility toggles
Before you start: Run python app.py from the src folder (with the processed network data generated). Then open this site at http://localhost:5000/.
